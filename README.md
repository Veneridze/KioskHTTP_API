# KioskHTTP_API
HTTP интерфейс для получения и отправки данных

##Методы получение данных

###**auth**

Параметры:
mode = auth
login = Логин
pass = Пароль

JSON - Ответ от сервера:

Если данные верны:
```json
{
  "status": "OK",
  "key": "abc123456789"
}
```

Если данные неверны
```json
{
  "status": "INCORRECT"
}
```
